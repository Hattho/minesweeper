Function that takes as input an empty board and two starting coordinates and generates it such that no bomb is placed close to the starting position.

Parameters
*boardThe board and all it's details contained in a struct.
start_lineThe line on which the starting position is located.
start_colThe column on which the starting position is located.

The function firstly makes sure it doesn't place the bomb on or adjecent to the starting position, then places it and increases the value of every square around it by one.
